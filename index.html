Joguinho

Um jogo sobre descobrir o númro secreto

Para isso, será necessário que o programa escolha um número secreto, e que o jogador interaja, tentando adivinhar o número. O jogador fará seus chutes, e o programa fornecerá dicas, indicando se o número é maior ou menor.

Principio de progamação utilizando JavaScript
